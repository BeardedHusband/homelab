# My Homelab

This repository documents the design, configuration, maintenance, and continuous improvement of my personal homelab.

The goal of this lab is to gain practical systems administration experience by building and maintaining services that I rely on every day.

---

# Objectives

- Build practical Linux administration experience
- Learn Docker and container management
- Improve networking knowledge
- Practice troubleshooting and disaster recovery
- Document everything
- Continuously improve infrastructure
